Installation Guide

Follow these simple steps to set up Odoo Direct Print

1

Download & Install

Visit mateusc7.github.io and download the new Odoo Direct Print - Agent V2. This version is required for Multi-POS and Dual Printing features.

2

Authorize Your PC

Click "Step 1: Authorize PC" in the agent. Accept the security warning to authorize communication with Odoo.

3

Confirm Activation

After authorization, you'll see "Printing service is active and authorized". The agent is now ready.

4

Configure in Odoo

In Odoo, go to Settings ‚Üí Odoo Direct Print. Copy the IP from the agent and paste it in Print Agent Configuration.

5

Assign Printers

Navigate to Report Management. Select a report and enter the printer name from the agent's list.

6

Configure POS (Optional)

For POS, go to settings ‚Üí Connected Devices ‚Üí Local Printing. Configure kitchen printers in Preparation ‚Üí Manage Printers.

Configuration Scenarios

üè∑Ô∏è How to Print ZPL Labels in Stock

Step 1: Configure the Report

Go to the "Report Management" menu and search for "zpl". Select the label report you want to use and assign your ZPL printer (e.g., Zebra, TSC) from the dropdown list.

Step 2: Print from Stock

Once configured, go to the Inventory/Stock module. Select products, click "Print Labels", and choose the layout (with price or without price). It will print directly to your label printer.

üìÑ How to Print PDF Invoices

Configuration

To enable direct PDF printing for invoices:

  1. Go to Report Management.
  2. Search for the report named "Invoice PDF".
  3. Select your standard A4/Letter printer.

Now, every time you click "Print Invoice", it will be sent directly to the printer.
